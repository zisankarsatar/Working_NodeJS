## Working_NodeJS

Node.JS&amp;MongoDB
Node.js ve MongoDB kullanrak temel işlemlerin olduğu bir uygulama.
Uygulamayı çalıştırmak için komut satırında uygulama yolunu vererek;

  "npm start"
 
 kullanılarak çalıştırılabilir.
